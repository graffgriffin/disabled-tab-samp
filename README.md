# Disabled TAB in GTA San Andreas Multiplayer

Disables the standard GUI Tab interface with a list of players on the server.

It is important to note that the main multiplayer module must be named "samp.dll" for the .asi plugin to work correctly.

I recommend using this plugin if you are creating your own GUI and want to remove the standard one or simply want to remove it from the game.
